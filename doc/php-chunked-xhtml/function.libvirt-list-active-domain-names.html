libvirt_list_active_domain_names

(php-libvirt 0.4)

libvirt_list_active_domain_names — Get active domain names

Description

array libvirt_list_active_domain_names ( resource $connect )

%COMMET

Parameters

connect

Connection resource of hypervisor.

Return Values

Return FALSE on fialure and an array of domain names for all domains running on the hypervisor.

See Also

  • libvirt_list_domain_names() - Get all domain names
  • libvirt_list_inactive_domain_names() - Get inactive domain names
