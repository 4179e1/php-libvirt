libvirt_network_is_active

(php-libvirt 0.4)

libvirt_network_is_active — Determine if the network is currently running

Description

integer libvirt_network_is_active ( resource $network )

Determine if the network is currently running

Parameters

network

Network resource, You can get it using various functions (e.g. libvirt_network_define_xml() or libvirt_network_lookup_by_name()).

Return Values

Return 1 if network is running, 0 if inactive, and -1 on failure.
