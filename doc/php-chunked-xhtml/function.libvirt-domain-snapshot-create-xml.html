libvirt_domain_snapshot_create_xml

(php-libvirt 0.4)

libvirt_domain_snapshot_create_xml — Creates a new snapshot of a domain based on the snapshot xml contained in xmlDesc.

Description

resource libvirt_domain_snapshot_create_xml ( resource $domain , string $xml )

Creates a new snapshot of a domain based on the snapshot xml contained in xmlDesc.

Parameters

domain

Domain to get snapshot count from. You can get domain resource using various functions (i.e. libvirt_domain_lookup_by_uuid() or libvirt_list_domains()).

xml

XML description of the snapshot. For more details see » http://www.libvirt.org/formatsnapshot.html.

Return Values

Return FALSE on failure and domain snapshot resource on success.

See Also

  • libvirt_list_domains() - List all domain resource
  • libvirt_domain_lookup_by_uuid() - Try to look up a domain by its UUID (binary)
  • libvirt_domain_lookup_by_uuid_string() - Try to look up a domain by its UUID (textual)
  • libvirt_domain_lookup_by_id() - Try to look up a domain based on the hypervisor ID number
  • libvirt_domain_lookup_by_name() - Try to look up a domain by its name.
  • libvirt_domain_snapshot_get_xml_desc() - Provide an XML description of the domain snapshot.
  • libvirt_domain_snapshot_delete() - Delete the snapshot
