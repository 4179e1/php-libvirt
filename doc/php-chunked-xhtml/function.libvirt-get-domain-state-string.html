libvirt_get_domain_state_string

(php-libvirt 0.4)

libvirt_get_domain_state_string — Return the string of a corresponding domain state number

Description

string libvirt_get_domain_state_string ( integer $state )

Return the string of a corresponding domain state number

Parameters

state

Domain state number, which is obtained from libvirt_get_domain_get_info().

Return Values

Return one of the following:

  • VIR_DOMAIN_NOSTATE
  • VIR_DOMAIN_RUNNING
  • VIR_DOMAIN_BLOCKED
  • VIR_DOMAIN_PAUSED
  • VIR_DOMAIN_SHUTDOWN
  • VIR_DOMAIN_SHUTOFF
  • VIR_DOAMIN_CRASHED
  • Err: Unknown State - if an invalid state number is given

Examples

Example #1 libvirt_get_domain_state_string() example

This exmaple show the usage of the function.

<?PHP
// omitted
$info = libvirt_domain_get_info ($domain);
echo (libvirt_get_domain_state_string ($info['state']);
?>

The above example will output something similar to:

VIR_DOAMIN_RUNNING
