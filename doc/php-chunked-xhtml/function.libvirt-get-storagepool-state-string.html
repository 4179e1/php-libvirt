libvirt_get_storagepool_state_string

(php-libvirt 0.4)

libvirt_get_storagepool_state_string — Return the string of a corresponding storagepool state number

Description

string libvirt_get_storagepool_state_string ( integer $state )

Return the string of a corresponding storagepool state number

Parameters

state

Storage pool state number, which is obtained from libivrt_storagepool_get_infof().

Return Values

One of the following:

  • VIR_STORAGE_POOL_INACTIVE
  • VIR_STORAGE_POOL_BUILDING
  • VIR_STORAGE_POOL_RUNNING
  • VIR_STORAGE_POOL_DEGRADED
  • VIR_STORAGE_POOL_INACCESSIBLE
  • Err: Unknown State - If an invalid state is given

Examples

Example #1 libvirt_list_active_storagepools() example

This example show the usage of the function.

<?PHP
$pool = libvirt_storage_pool_lookup_by_name ($conn, "default");
$info = libvirt_storage_pool_get_info ($pool);
echo (libvirt_get_storagepool_state_string ($info['state']);
?>

The above example will output something similar to:

VIR_STORAGE_POOL_INACTIVE
