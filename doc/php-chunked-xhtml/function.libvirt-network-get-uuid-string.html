libvirt_network_get_uuid_string

(php-libvirt 0.4)

libvirt_network_get_uuid_string — Get the UUID for a network as string.

Description

string libvirt_network_get_uuid_string ( resource $network )

Get the UUID for a network as string. For more information about UUID see RFC4122.

Parameters

network

Network resource, You can get it using various functions (e.g. libvirt_network_define_xml() or libvirt_network_lookup_by_name()).

Return Values

Return the UUID string of the network, or FALSE on failure.
