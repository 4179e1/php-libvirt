libvirt_get_active_storagepool_count

(php-libvirt 0.4)

libvirt_get_active_storagepool_count — Get the number of active storagepools

Description

integer libvirt_get_active_storagepool_count ( resource $connect )

Provides the number of active storagepools.

Parameters

connect

Connection resource of hypervisor.

Return Values

Return FALSE on failue and the number of actived storagepools on success;

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_get_storagepool_count() - Get the number of all storagepools
  • libvirt_get_inactive_storagepool_count() - Get the number of inactived storagepools
