libvirt_node_device_destroy

(php-libvirt 0.4)

libvirt_node_device_destroy — Destroy the device object.

Description

bool libvirt_node_device_destroy ( resource $device )

Destroy the device object. The virtual device is removed from the host operating system. This function may require privileged access

Parameters

device

Node device resource, you can get it using libvirt_list_node_devices() or libvirt_node_device_lookup_by_name().

Return Values

Return TRUE on success and FALSE on failure.

See Also

  • libvirt_node_device_lookup_by_name() - Lookup a node device by its name.
  • libvirt_list_node_devices() - List all node devices
