libvirt_list_inactive_interfaces

(php-libvirt 0.4)

libvirt_list_inactive_interfaces — List active interfaces

Description

array libvirt_list_inactive_interfaces ( resource $connect )

List active interfaces.

Parameters

connect

Connection resource of hypervisor.

Return Values

Return an array contains the name of all inactive interfaces, or FALSE on failure.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_list_interfaces() - List all interfaces
  • libvirt_list_active_interfaces() - List active interfaces
