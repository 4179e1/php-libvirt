libvirt_list_networks

(php-libvirt 0.4)

libvirt_list_networks — List all networks

Description

integer libvirt_list_networks ( resource $connect )

List all networks, include actived and inactived.

Parameters

connect

Connection resource of hypervisor.

Return Values

Return an array that contains the name of all networks, or FALSE on failure.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_list_active_networks() - List active networks
  • libvirt_list_inactive_networks() - List inactive networks
  • libvirt_network_lookup_by_name() - Try to lookup a network on the given hypervisor based on its name.
