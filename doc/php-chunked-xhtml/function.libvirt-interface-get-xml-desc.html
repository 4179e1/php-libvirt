libvirt_interface_get_xml_desc

(php-libvirt 0.4)

libvirt_interface_get_xml_desc — Provide an XML description of the interface.

Description

string libvirt_interface_get_xml_desc ( resource $interface [, integer $flags = 0 ] )

Provide an XML description of the interface.

Parameters

interface

Interface resource, you can get it using various functions (e.g. libvirt_list_interfaces() or libvirt_interface_lookup_by_name()).

flags

Logic OR of any of these flags:

  • VIR_INTERFACE_XML_INACTIVE

Return Values

Return the xml description of the interface, or FALSE on failure.

See Also

  • libvirt_interface_define_xml() - Define an interface (or modify existing interface configuration)
