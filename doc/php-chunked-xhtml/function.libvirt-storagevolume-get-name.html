libvirt_storagevolume_get_name

(php-libvirt 0.4)

libvirt_storagevolume_get_name — Fetch the storage volume name

Description

string libvirt_storagevolume_get_name ( resource $volume )

Fetch the storage volume name. This is unique within the scope of a pool.

Parameters

volume

Storagevolume resource, which obtained by libvirt_list_volumes().

Return Values

Return the name of the storagevolume, or FALSE on failure.

See Also

  • libvirt_list_volumes() - List all volumes in a storagepool
