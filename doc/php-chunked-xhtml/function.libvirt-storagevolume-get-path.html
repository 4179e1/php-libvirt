libvirt_storagevolume_get_path

(php-libvirt 0.4)

libvirt_storagevolume_get_path — Fetch the storage volume path.

Description

string libvirt_storagevolume_get_path ( resource $volume )

Fetch the storage volume path. Depending on the pool configuration this is either persistent across hosts, or dynamically assigned at pool startup. Consult pool documentation for information on getting the persistent naming.

Parameters

volume

Storagevolume resource, which obtained by libvirt_list_volumes().

Return Values

Return the path of a storagevolume, or FALSE on failue.

See Also

  • libvirt_list_volumes() - List all volumes in a storagepool
