libvirt_network_get_xml_desc

(php-libvirt 0.4)

libvirt_network_get_xml_desc — Provide an XML description of the network.

Description

string libvirt_network_get_xml_desc ( resource $network )

Provide an XML description of the network.

Parameters

network

Network resource, You can get it using various functions (e.g. libvirt_network_define_xml() or libvirt_network_lookup_by_name()).

Return Values

Return the XML description of the network, or FALSE on failure. For more details, see » http://libvirt.org/formatnetwork.html.
