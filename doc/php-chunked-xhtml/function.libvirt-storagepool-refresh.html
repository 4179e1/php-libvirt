libvirt_storagepool_refresh

(php-libvirt 0.4)

libvirt_storagepool_refresh — Request that the pool refresh its list of volumes.

Description

bool libvirt_storagepool_refresh ( resource $pool )

Request that the pool refresh its list of volumes. This may involve communicating with a remote server, and/or initializing new devices at the OS layer.

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return TRUE on success and FALSE on failure.
