libvirt_storagepool_get_volume_count

(php-libvirt 0.4)

libvirt_storagepool_get_volume_count — Fetch the number of storage volumes within a pool

Description

integer libvirt_storagepool_get_volume_count ( resource $pool )

Fetch the number of storage volumes within a pool.

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return the number of storage volumes within the pool, or FALSE on failure.
