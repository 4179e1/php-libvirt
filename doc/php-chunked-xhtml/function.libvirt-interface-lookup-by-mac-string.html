libvirt_interface_lookup_by_mac_string

(php-libvirt 0.4)

libvirt_interface_lookup_by_mac_string — Try to lookup an interface on the given hypervisor based on its MAC.

Description

resource libvirt_interface_lookup_by_mac_string ( resource $connect , string $mac_string )

Try to lookup an interface on the given hypervisor based on its MAC.

Parameters

connect

Connection resource of hypervisor.

mac_string

the MAC for the interface

Return Values

Return interface resource, or FALSE on failure.

See Also

  • libvirt_interface_get_mac_string() - Get the MAC for an interface as string.
