libvirt_interface_lookup_by_name

(php-libvirt 0.4)

libvirt_interface_lookup_by_name — Try to lookup an interface on the given hypervisor based on its name.

Description

resource libvirt_interface_lookup_by_name ( resource $connect , name $name )

Try to lookup an interface on the given hypervisor based on its name.

Parameters

connect

Connection resouce of hypervisor.

name

Name of the interface.

Return Values

Return interface resource, or FALSE on failure.

  • libvirt_interface_get_name() - Get the public name for that interface
