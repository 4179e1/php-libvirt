libvirt_interface_define_xml

(php-libvirt 0.4)

libvirt_interface_define_xml — Define an interface (or modify existing interface configuration)

Description

bool libvirt_interface_define_xml ( resource $connect , string $xml )

Define an interface (or modify existing interface configuration)

Parameters

connect

Connection resource of hypervisor.

xml

XML description of the interface.

Return Values

Return interface resource, or FALSE on failure.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_interface_undefine() - Undefine an interface, ie remove it from the config.
  • libvirt_interface_get_xml_desc() - Provide an XML description of the interface.
