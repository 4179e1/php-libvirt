libvirt_list_inactive_domains

(php-libvirt 0.1)

libvirt_list_inactive_domains — Get inactive domain resources

Description

array libvirt_list_inactive_domains ( resource $connection )

Get the resources of defined but inactive domains.

Parameters

connection

Connection resource of hypervisor.

Return Values

Return FALSE on fialure and an array of domain resources for all inactive domains on the hypervisor.
