libvirt_interface_destroy

(php-libvirt 0.4)

libvirt_interface_destroy — deactivate an interface (ie call "ifdown")

Description

bool libvirt_interface_destroy ( resource $interface )

deactivate an interface (ie call "ifdown")

Parameters

interface

Interface resource, you can get it using various functions (e.g. libvirt_list_interfaces() or libvirt_interface_lookup_by_name()).

Return Values

Return TRUE on success and FALSE on failure.

See Also

  • libvirt_list_interfaces() - List all interfaces
  • libvirt_interface_create_xml()
  • libvirt_interface_lookup_by_name() - Try to lookup an interface on the given hypervisor based on its name.
  • libvirt_interface_lookup_by_mac_string() - Try to lookup an interface on the given hypervisor based on its MAC.
  • libvirt_interface_create() - Activate an interface (ie call "ifup")
