libvirt_domain_get_autostart

(php-libvirt 0.4)

libvirt_domain_get_autostart — Whether the domian is configured to be autostart

Description

integer libvirt_domain_get_autostart ( resource $domain )

Return a integer indicating whether the domain configured to be automatically started when the host machine boots.

Parameters

domain

Domain to get autostart state. You can get domain resource using various functions (i.e. libvirt_domain_lookup_by_uuid() or libvirt_list_domains()).

Return Values

Return 1 if autostat, 0 if no autostart, and FALSE on failure

See Also

  • libvirt_list_domains() - List all domain resource
  • libvirt_domain_lookup_by_uuid() - Try to look up a domain by its UUID (binary)
  • libvirt_domain_lookup_by_uuid_string() - Try to look up a domain by its UUID (textual)
  • libvirt_domain_lookup_by_id() - Try to look up a domain based on the hypervisor ID number
  • libvirt_domain_lookup_by_name() - Try to look up a domain by its name.
  • libvirt_domain_create_xml() - Launch a new guest domain, based on an XML description
  • libvirt_domain_define_xml() - Define a domain, but does not start it.
