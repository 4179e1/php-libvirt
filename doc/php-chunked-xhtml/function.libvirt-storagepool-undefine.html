libvirt_storagepool_undefine

(php-libvirt 0.4)

libvirt_storagepool_undefine — Undefine a storagepool

Description

bool libvirt_storagepool_undefine ( resource $pool [, %PARAM2_TYPE $%PARAM2_NAME = %INIT_VALUE ] )

Undefine a storagepool but does not stop it if it is running.

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return TRUE on success and FALSE on failure.

See Also

  • libvirt_storagepool_define_xml() - Define a new inactive storage pool based on its XML description.
