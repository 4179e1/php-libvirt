libvirt_storagepool_lookup_by_uuid_string

(php-libvirt 0.4)

libvirt_storagepool_lookup_by_uuid_string — Try to look up a storagepool by its UUID (textual)

Description

resource libvirt_storagepool_lookup_by_uuid_string ( resource $connect , string $uuid_string )

Try to look up a domain on the given hypervisor based on its UUID (in textual representation).

Parameters

connect

Connection resource of hypervisor.

uuid_string

UUID of a storagepool to look up (in textual representation).

Return Values

Return the storagepool resource, or FALSE on failure.
