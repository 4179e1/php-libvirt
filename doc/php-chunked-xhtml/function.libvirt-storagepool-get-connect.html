libvirt_storagepool_get_connect

(php-libvirt 0.4)

libvirt_storagepool_get_connect — Returns connection resource for a storagepool

Description

resource libvirt_storagepool_get_connect ( resource $pool )

Returns connection resource for a storagepool

Note: Notice

. You should not rely on this function. It is better to keep your own reference of the connections and storagepools.

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return FALSE on failure and storagepool resource on success;
