libvirt_interface_is_active

(php-libvirt 0.4)

libvirt_interface_is_active — Determine if the interface is currently running

Description

integer libvirt_interface_is_active ( resource $interface )

Determine if the interface is currently running

Parameters

interface

Interface resource, you can get it using various functions (e.g. libvirt_list_interfaces() or libvirt_interface_lookup_by_name()).

Return Values

Return 1 if interface is running, 0 if inactive, and -1 on failure.

See Also

  • libvirt_list_interfaces() - List all interfaces
  • libvirt_interface_lookup_by_name() - Try to lookup an interface on the given hypervisor based on its name.
  • libvirt_interface_lookup_by_mac_string() - Try to lookup an interface on the given hypervisor based on its MAC.
