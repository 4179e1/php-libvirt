libvirt_get_type

(php-libvirt 0.4)

libvirt_get_type — Get the name of the Hypervisor software used.

Description

string libvirt_get_type ( resource $connect )

Get the name of the Hypervisor software used.

Parameters

connect

Connection resource of hypervisor, obtained from libvirt_connect().

Return Values

Return FALSE on failure and the hypervisor's name on success.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_get_max_vcpus() - Provides the maximum number of virtual CPUs supported for a guest VM of a specific type.
