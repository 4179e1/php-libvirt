libvirt_network_define_xml

(php-libvirt 0.4)

libvirt_network_define_xml — Define a network, but does not create it

Description

resource libvirt_network_define_xml ( resource $connect , string $xml )

Define a network, but does not create it

Parameters

connect

Connection resource of hypervisor.

xml

XML description of the network. For more detail, see » http://libvirt.org/formatnetwork.html

Return Values

Return the resource of the network, or FALSE on failure.
