libvirt_list_active_storagepools

(php-libvirt 0.4)

libvirt_list_active_storagepools — List actived storagepools

Description

array libvirt_list_active_storagepools ( resource $connect )

This function return an array of actived storagepool names for the connection resource.

Parameters

connect

Connection resource of hypervisor;

Return Values

Return FALSE on failure and an array of actived storagepool names on success

  • libvirt_list_storagepools() - List all storagepools
