libvirt_list_ininactive_storagepools — List inactived storagepools

Description

array libvirt_list_ininactive_storagepools ( interger $connect )

This function return an array of inactived storagepool names for the connection resource.

Parameters

%PARAM_NAME

Connection resource of hypervisor;

Return Values

Return FALSE on failure and an array of inactived storagepool names on success

See Also

  • libvirt_list_storagepools() - List all storagepools
  • libvirt_list_active_storagepools() - List actived storagepools
