libvirt_get_max_vcpus

(php-libvirt 0.4)

libvirt_get_max_vcpus — Provides the maximum number of virtual CPUs supported for a guest VM of a specific type.

Description

integer libvirt_get_max_vcpus ( resource $connect , string $type )

Provides the maximum number of virtual CPUs supported for a guest VM of a specific type. The 'type' parameter here corresponds to the 'type' attribute in the domain element of the XML (e.g. "xen").

Parameters

connect

Connection resource of hypervisor, obtained from libvirt_connect().

type

The 'type' parameter here corresponds to the 'type' attribute in the domain element of the XML (e.g. "xen"). You can get type by libvirt_get_type().

Return Values

Return FALSE on failure and the max virtual cpu number that the VM support on success;

Examples

Example #1 libvirt_get_max_vcpus() example

This example show the max virtual cpus that xen VM support.

<?PHP
$conn = libvirt_connect("xen:///");
$type = libvirt_get_type($conn);
echo ("Max vcpus: " . libvirt_get_max_vcpus ($conn, $type));
?>

The above example will output something similar to:

Max vcpus: 32
