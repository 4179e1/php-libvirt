libvirt_storagepool_destroy

(php-libvirt 0.4)

libvirt_storagepool_destroy — Destroy an active storage pool

Description

integer libvirt_storagepool_destroy ( resource $pool )

Destroy an active storage pool. This will deactivate the pool on the host, but keep any persistent config associated with it. If it has a persistent config it can later be restarted with libvirt_storagepool_create().

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return FALSE on failure and storagepool resource on success.

See Also

  • libvirt_storagepool_define_xml() - Define a new inactive storage pool based on its XML description.
  • libvirt_storagepool_define_xml() - Define a new inactive storage pool based on its XML description.
  • libvirt_storagepool_lookup_by_name() - Lookup storagepool resource by name
  • libvirt_storagepool_lookup_by_uuid_string() - Try to look up a storagepool by its UUID (textual)
  • libvirt_storagepool_create() - Starts an inactive storage pool
