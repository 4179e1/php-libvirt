libvirt_list_node_devices

(php-libvirt 0.4)

libvirt_list_node_devices — List all node devices

Description

array libvirt_list_node_devices ( resource $connect )

List all node devices

Parameters

connect

Connection resource of hypervisor.

Return Values

Return an array contains all device names in a node, or FALSE on failure.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_get_node_device_count() - Get the number of node devices
