libvirt_get_uri

(php-libvirt 0.4)

libvirt_get_uri — This returns the URI (name) of the hypervisor connection.

Description

string libvirt_get_uri ( resource $connect )

This returns the URI (name) of the hypervisor connection. Normally this is the same as or similar to the string passed to the libvirt_connection(), but the driver may make the URI canonical. If 'url' was omitted in libvirt_connection(), then the driver will return a non-NULL URI which can be used to connect to the same hypervisor later.

Parameters

connect

Connection resource of hypervisor, obtained from libvirt_connect().

Return Values

Return FALSE on failure and the uri of the connection on success;
