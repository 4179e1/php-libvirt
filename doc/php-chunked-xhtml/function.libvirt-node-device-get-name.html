libvirt_node_device_get_name

(php-libvirt 0.4)

libvirt_node_device_get_name — Just return the device name

Description

string libvirt_node_device_get_name ( resource $device )

Just return the device name.

Parameters

device

Node device resource, you can get it using libvirt_list_node_devices() or libvirt_node_device_lookup_by_name().

Return Values

Return the name of the device, or FALSE on failure.

See Also

  • libvirt_node_device_lookup_by_name() - Lookup a node device by its name.
  • libvirt_list_node_devices() - List all node devices
