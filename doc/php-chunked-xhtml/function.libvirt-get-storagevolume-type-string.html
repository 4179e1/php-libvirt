libvirt_get_storagevolume_type_string

(php-libvirt 0.4)

libvirt_get_storagevolume_type_string — Return the stirng of a corresponding storagevolume type number

Description

string libvirt_get_storagevolume_type_string ( integer $type )

Return the stirng of a corresponding storagevolume type number

Parameters

type

Storagevolume type number, which is obtained from libvirt_get_storagevolume_info().

Return Values

Return one of the following:

  • VIR_STORAGE_VOL_FILE
  • VIR_STORAGE_VOL_BLOCK
  • Err: Unknown Type - if an invalid state number is given

Examples

Example #1 libvirt_get_storagevolume_type_string() example

This example show the usage of the function

<?PHP
    $vol = libvirt_storagevolume_lookup_by_name ($conn, "vm-vol");
    $info = libvirt_storagevolume_get_info ($vol);
    echo (libvirt_get_storagevolume_type_string ($info['type']);
?>

The above example will output something similar to:

VIR_STORAGE_VOL_FILE
