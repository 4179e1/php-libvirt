libvirt_network_set_autostart

(php-libvirt 0.4)

libvirt_network_set_autostart — Configure the network to be automatically started when the host machine boots.

Description

bool libvirt_network_set_autostart ( resource $network )

Configure the network to be automatically started when the host machine boots.

Parameters

network

Network resource, You can get it using various functions (e.g. libvirt_network_define_xml() or libvirt_network_lookup_by_name()).

Return Values

Return TRUE on success and FALSE on failure.
