libvirt_domain_snapshot_lookup_by_name

(php-libvirt 0.4)

libvirt_domain_snapshot_lookup_by_name — Try to lookup a domain snapshot based on its name.

Description

resource libvirt_domain_snapshot_lookup_by_name ( resource $domain , string $name )

Try to lookup a domain snapshot based on its name.

Parameters

domain

Domain to lookup snapshots. You can get domain resource using various functions (i.e. libvirt_domain_lookup_by_uuid() or libvirt_list_domains()).

name

Name of the snapshot to lookup. You can get domain snapshot names by using libvirt_libvirt_domain_snapshots() .

Return Values

Return FALSE on failure, domain snapshot resource on success, or NULL if domain don't have any snapshots.

See Also

  • libvirt_list_domains() - List all domain resource
  • libvirt_domain_lookup_by_uuid() - Try to look up a domain by its UUID (binary)
  • libvirt_domain_lookup_by_uuid_string() - Try to look up a domain by its UUID (textual)
  • libvirt_domain_lookup_by_id() - Try to look up a domain based on the hypervisor ID number
  • libvirt_domain_lookup_by_name() - Try to look up a domain by its name.
  • libvirt_list_domain_snapshots() - List all snapshot of the domain
