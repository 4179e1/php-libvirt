libvirt_list_interfaces

(php-libvirt 0.4)

libvirt_list_interfaces — List all interfaces

Description

integer libvirt_list_interfaces ( resource $connect )

List all interfaces, include active and inactive.

Parameters

connect

Conenction resource of hypervisor.

Return Values

Return an array contains the name of all interfaces, or FALSE on failure.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_list_active_interfaces() - List active interfaces
  • libvirt_list_inactive_interfaces() - List active interfaces
