libvirt_storagepool_define_xml

(php-libvirt 0.4)

libvirt_storagepool_define_xml — Define a new inactive storage pool based on its XML description.

Description

resource libvirt_storagepool_define_xml ( resource $connect , string $xml )

Define a new inactive storage pool based on its XML description. The pool is persistent, until explicitly undefined.

Parameters

connect

Connection resource of hypervisor.

xml

XML description of the storagepool. For more detail, see » http://libvirt.org/formatstorage.html.

Return Values

Return FALSE on failure and Storagepool resource on success;

See Also

  • libvirt_storagepool_create() - Starts an inactive storage pool
  • libvirt_storagepool_undefine() - Undefine a storagepool
