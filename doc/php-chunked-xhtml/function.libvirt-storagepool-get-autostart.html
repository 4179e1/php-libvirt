libvirt_storagepool_get_autostart

(php-libvirt 0.4)

libvirt_storagepool_get_autostart — Whether the storagepool is configured to be autostart

Description

integer libvirt_storagepool_get_autostart ( resource $pool )

Return a integer indicating whether the storagepool configured to be automatically started when the host machine boots.

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return 1 if autostat, 0 if no autostart, and FALSE on failure

See Also

  • libvirt_storagepool_set_autostart() - Configure the domain to be automatically started when the host machine boots.
  • libvirt_storagepool_lookup_by_name() - Lookup storagepool resource by name
  • libvirt_storagepool_lookup_by_uuid_string() - Try to look up a storagepool by its UUID (textual)
  • libvirt_list_storagepools() - List all storagepools
