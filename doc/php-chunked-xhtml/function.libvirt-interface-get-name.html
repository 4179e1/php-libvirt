libvirt_interface_get_name

(php-libvirt 0.4)

libvirt_interface_get_name — Get the public name for that interface

Description

string libvirt_interface_get_name ( resource $interface )

Get the public name for that interface

Parameters

interface

Interface resource, you can get it using various functions (e.g. libvirt_list_interfaces() or libvirt_interface_lookup_by_name()).

Return Values

Return the name of the interface, or FALSE on failure.

See Also

  • libvirt_interface_lookup_by_name() - Try to lookup an interface on the given hypervisor based on its name.
