libvirt_storagepool_get_uuid_string

(php-libvirt 0.4)

libvirt_storagepool_get_uuid_string — Get the UUID for a storagepool as string

Description

string libvirt_storagepool_get_uuid_string ( resource $pool )

Get the UUID for a domain as string (i.e. 25ab2490-7c4c-099f-b647-45ff8efa73f6 ). For more information about UUID see RFC4122.

Parameters

pool

Storagepool resource, you can get it using various functions (e.g. libvirt_storagepool_define_xml(), libvirt_storagepool_lookup_by_name()).

Return Values

Return the UUID string of storagepool or FALSE on failure.

See Also

  • libvirt-storagepool-lookup-by-uuid-string() - Try to look up a storagepool by its UUID (textual)
