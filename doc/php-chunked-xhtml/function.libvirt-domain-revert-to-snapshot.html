libvirt_domain_revert_to_snapshot

(php-libvirt 0.4)

libvirt_domain_revert_to_snapshot — Revert the domain to a given snapshot.

Description

bool libvirt_domain_revert_to_snapshot ( resource $snapshot )

Revert the domain to a given snapshot.

Parameters

snapshot

The snapshot to revert to. You can get snapshot resource using various functions(i.e. libvirt_list_domain_snapshots() or libvirt_domain_snapshot_lookup_by_name()) .

Return Values

Return TRUE on success and FALSE on failure.

See Also

  • libvirt_list_domain_snapshots() - List all snapshot of the domain
  • libvirt_domain_snapshot_lookup_by_name() - Try to lookup a domain snapshot based on its name.
  • libvirt_domain_snapshot_count() - Provides the number of domain snapshots for this domain.
