libvirt_get_active_interface_count

(php-libvirt 0.4)

libvirt_get_active_interface_count — Get the number of inactive interfaces

Description

integer libvirt_get_active_interface_count ( resource $connect )

Get the number of inactive interfaces

Parameters

connect

Connection resource of hypervisor.

Return Values

Return the number of active interfaces, or FALSE on failure.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
  • libvirt_get_interface_count() - Get the number of all interfaces.
  • libvirt_get_inactive_interface_count() - Get the number of all inactive interfaces
