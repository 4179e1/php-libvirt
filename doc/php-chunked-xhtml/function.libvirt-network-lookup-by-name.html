libvirt_network_lookup_by_name

(php-libvirt 0.4)

libvirt_network_lookup_by_name — Try to lookup a network on the given hypervisor based on its name.

Description

resource libvirt_network_lookup_by_name ( resource $connect , string $name )

Try to lookup a network on the given hypervisor based on its name.

Parameters

network

Connection resource of hypervisor.

uuid_string

name for the network

Return Values

Return the network resource, or FALSE on failure.
