libvirt_network_get_autostart

(php-libvirt 0.4)

libvirt_network_get_autostart — Whether the network si configured to be autostart

Description

integer libvirt_network_get_autostart ( resource $network )

Return a integer indicating whether the network configured to be automatically started when the host machine boots.

Parameters

network

Network resource, You can get it using various functions (e.g. libvirt_network_define_xml() or libvirt_network_lookup_by_name()).

Return Values

Return 1 if autostat, 0 if no autostart, and FALSE on failure
