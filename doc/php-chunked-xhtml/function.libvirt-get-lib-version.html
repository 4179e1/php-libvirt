libvirt_get_lib_version

(php-libvirt 0.4)

libvirt_get_lib_version — Return the version of libvirt used by the daemon running on the connected host.

Description

integer libvirt_get_lib_version ( resource $connect )

Return the version of libvirt used by the daemon running on the connected host.

Parameters

connect

Connection resource of hypervisor, obtained from libvirt_connect().

Return Values

Return FALSE on failure and version of the daemon that connecting the hypervisor on success.

See Also

  • libvirt_connect() - Get a connection to the Hypervisor
