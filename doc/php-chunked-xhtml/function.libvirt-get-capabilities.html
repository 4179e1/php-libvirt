libvirt_get_capabilities

(php-libvirt 0.4)

libvirt_get_capabilities — Provides capabilities of the hypervisor / driver.

Description

string libvirt_get_capabilities ( resource $connect )

Provides capabilities of the hypervisor / driver.

Parameters

connect

Connection Resource of hypervisor, obtained from libvirt_connect().

Return Values

Return FALSE on failure and an XML defining the capabilities. For more details, see: » http://libvirt.org/formatcaps.html
