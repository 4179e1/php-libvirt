libvirt_network_get_connect

(php-libvirt 0.4)

libvirt_network_get_connect — Returns connection resource for a network

Description

resource libvirt_network_get_connect ( resource $network )

Returns the connection resource for a network.

Note: Notice

. You should not rely on this function. It is better to keep your own reference of the connections and networks.

Parameters

network

Network resource, You can get it using various functions (e.g. libvirt_network_define_xml() or libvirt_network_lookup_by_name()).

Return Values

Return the connection resource of the network, or FALSE on failue.
