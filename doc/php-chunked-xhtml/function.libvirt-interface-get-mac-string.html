libvirt_interface_get_mac_string

(php-libvirt 0.4)

libvirt_interface_get_mac_string — Get the MAC for an interface as string.

Description

string libvirt_interface_get_mac_string ( resource $interface )

Get the MAC for an interface as string. For more information about MAC see RFC4122.

Parameters

interface

Interface resource, you can get it using various functions (e.g. libvirt_list_interfaces() or libvirt_interface_lookup_by_name()).

Return Values

Return mac string of the interface, or FALSE on failure.

See Also

  • libvirt-interface-lookup-by-mac-string() - Try to lookup an interface on the given hypervisor based on its MAC.
