libvirt_domain_has_current_snapshot

(php-libvirt 0.4)

libvirt_domain_has_current_snapshot — Determine if the domain has a current snapshot.

Description

integer libvirt_domain_has_current_snapshot ( resource $domain )

Determine if the domain has a current snapshot.

Parameters

domain

Whether this domain has a current snapshot. You can get domain resource using various functions (i.e. libvirt_domain_lookup_by_uuid() or libvirt_list_domains()).

Return Values

1 if such snapshot exists, 0 if it doesn't, -1 on error.

See Also

  • libvirt_list_domains() - List all domain resource
  • libvirt_domain_lookup_by_uuid() - Try to look up a domain by its UUID (binary)
  • libvirt_domain_lookup_by_uuid_string() - Try to look up a domain by its UUID (textual)
  • libvirt_domain_lookup_by_id() - Try to look up a domain based on the hypervisor ID number
  • libvirt_domain_lookup_by_name() - Try to look up a domain by its name.
  • libvirt_domain_create_xml() - Launch a new guest domain, based on an XML description
  • libvirt_domain_define_xml() - Define a domain, but does not start it.
  • libvirt_list_domain_snapshots() - List all snapshot of the domain
  • libvirt_domain_snapshot_lookup_by_name() - Try to lookup a domain snapshot based on its name.
  • libvirt_domain_snapshot_count() - Provides the number of domain snapshots for this domain.
  • libvirt_domain_snapshot_current() - Get the current snapshot for a domain, if any.
