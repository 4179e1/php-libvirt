libvirt_interface_undefine

(php-libvirt 0.4)

libvirt_interface_undefine — Undefine an interface, ie remove it from the config.

Description

bool libvirt_interface_undefine ( resource $interface )

Undefine an interface, ie remove it from the config.

Parameters

interface

Interface resource, you can get it using various functions (e.g. libvirt_list_interfaces() or libvirt_interface_lookup_by_name()).

Return Values

Return TRUE on success and FALSE on failure.
