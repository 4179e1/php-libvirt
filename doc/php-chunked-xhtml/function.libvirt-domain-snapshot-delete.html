libvirt_domain_snapshot_delete

(php-libvirt 0.4)

libvirt_domain_snapshot_delete — Delete the snapshot

Description

bool libvirt_domain_snapshot_delete ( resource $snapshot [, integer $flags = 0 ] )

Delete the snapshot. If @flags is 0, then just this snapshot is deleted, and changes from this snapshot are automatically merged into children snapshots. If flags is VIR_DOMAIN_SNAPSHOT_DELETE_CHILDREN, then this snapshot and any children snapshots are deleted.

Parameters

snapshot

The snapshot to delete. You can get snapshot resource using various functions(i.e. libvirt_list_domain_snapshots() or libvirt_domain_snapshot_lookup_by_name()) .

flags

Logical OR of any of these flags:

  • VIR_DOMAIN_SNAPSHOT_DELETE_CHILDEREN

Return Values

Return TRUE on success and FALSE on failure.

See Also

  • libvirt_list_domain_snapshots() - List all snapshot of the domain
  • libvirt_domain_snapshot_lookup_by_name() - Try to lookup a domain snapshot based on its name.
  • libvirt_domain_snapshot_count() - Provides the number of domain snapshots for this domain.
  • libvirt_domain_snapshot_create_xml() - Creates a new snapshot of a domain based on the snapshot xml contained in xmlDesc.
